ゼロ初期化コードはモデル 'PathFollowing' 用に最適化されています。

整数のグローバル変数の初期値は 0 です True

仮定が正しくない場合は、[コンフィギュレーション パラメーター] > [コード生成] > [最適化] 設定で最適化を削除します。あるいは、仮定が有効になるようターゲット環境を構成します。

浮動小数点数

コード生成は浮動小数点数をサポートするように構成されているため、ターゲット環境では非正規数の計算が行われる可能性があります。

flush-to-zero で計算された非正規値 (FTZ) False
flush-to-zero の入力非正規値 (DAZ) False

仮定が正しくない場合、非正規数によってモデルと生成コードのシミュレーション間で結果の不一致が生じる可能性があります。詳細については、「非正規数のパフォーマンス」を参照してください。
